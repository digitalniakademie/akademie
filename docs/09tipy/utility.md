# Utility

Užitečné programy, které nemusí přímo souviset s grafikou.

## Linux
### hardinfo
Informace o systému.
### bpytop
Systémový monitor.

## Windows

### Ventoy
Spouštění live .iso operačních systémů
### Oracle VM VirtualBox
Virtualizace operačních systémů.
